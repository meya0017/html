Web Development Terminologies

Web Development Terminologies

HTML
HyperText Markup Language used to structure content on the web.
CSS
Cascading Style Sheets used for styling HTML elements.
JavaScript
A programming language used to make web pages interactive.
DOM
Document Object Model, represents the structure of a web page as a tree of objects.
API
Application Programming Interface, allows communication between software components.
HTTP
HyperText Transfer Protocol, the foundation of data communication on the web.
HTTPS
Secure version of HTTP that encrypts data for security.
URL
Uniform Resource Locator, the address used to access a resource on the internet.
Frontend
The part of a website that users interact with directly.
Backend
The server-side logic and database interaction of a web application.
Flexbox
A CSS layout module that makes it easy to align and distribute space among items in a container.
Grid
A CSS layout system that uses rows and columns to arrange elements.
Git
A version control system used to track changes in code during development.
Repository
A storage location for project code, often hosted on platforms like GitHub or GitLab.
Bootstrap
A popular CSS framework for building responsive and mobile-first websites quickly.
Semantic HTML
HTML that uses meaningful tags to describe content structure (e.g., <article>, <section>).
SEO
Search Engine Optimization, the practice of improving a website to rank higher in search engines.
Responsive Design
Design that adapts to different screen sizes and devices.
Meta Tag
Provides metadata about the HTML document, such as description or viewport settings.
Media Query
A CSS technique used to apply styles based on screen size or device type.
